  Solution: Hard refresh (Ctrl+Shift+R), clear cache (Ctrl+Shift+Delete)

  Problem: npm errors
  Solution: rm -r node_modules package-lock.json && npm install

  Problem: Java version errors
  Solution: Install Java 17+ and restart terminal
